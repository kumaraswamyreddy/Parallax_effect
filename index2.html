TYPES OF LISTS

HTML lists allow web developers to group a set of related items in lists. An unordered list starts with the ul tag. Each list item starts with the li tag. An ordered list starts with the ol tag. Each list item starts with the li tag. HTML also supports description lists. A description list is a list of terms, with a description of each term.
